## Local Preview
Just open `index.html` in a browser. No tooling required.

## Deploy
- **Netlify:** Repo → New site from Git → publish = `.` (see `netlify.toml`)
- **GitHub Pages:** Settings → Pages → Deploy from branch (root)

## Update Pointers
- **Email rendering:** managed in `script.js` (obfuscation + mailto).
- **Styling:** `styles.css` (cards, grid, dark theme).
- **Resume:** replace `resume.pdf` with a new export while keeping the filename.
